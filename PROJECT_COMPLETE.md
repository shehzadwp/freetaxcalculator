## 🚀 Quick Start (5 Minutes)

### 1. Install Dependencies
```bash
npm install
```

### 2. Configure Environment
Update `.env.local`:
```
NEXT_PUBLIC_GOOGLE_ADSENSE_ID=ca-pub-your-id
NEXT_PUBLIC_SITE_URL=https://yourdomain.com
```

### 3. Run Development Server
```bash
npm run dev
```

### 4. Open in Browser
Navigate to [http://localhost:3000](http://localhost:3000)

### 5. Deploy
Choose your platform from DEPLOYMENT.md and follow instructions

## 🎯 Getting Started Workflow

1. **Clone/Download**
   ```bash
   cd calculator\ webiste
   ```

2. **Install**
   ```bash
   npm install
   ```

3. **Configure**
   - Edit `.env.local`
   - Update domain in `lib/config/siteConfig.ts`
   - Add social links

4. **Customize**
   - Update branding in components
   - Change colors in tailwind.config.js
   - Update tax data if needed

5. **Test**
   ```bash
   npm run dev
   ```

6. **Build**
   ```bash
   npm run build
   npm start
   ```

7. **Deploy**
